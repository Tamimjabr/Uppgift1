Frågor om HTML5

1. Beskriv kort vad följande är:

  1. Element
  2. Tagg
  3. Attribut

Svar:

Element
det är komponenterna som HTML-documentet består av.
Tagg
Det är koden som beskriver var HTMLs element börjar och slutar och vilken uppgift detta element utför, samt det innehåller också elementets attribut ifall elementet har ett eller några.
Attribut
Det är egenskap som ger information om elementet eller påverkar på något sätt elementets beteende. Exempel: <img src=”bananan.jpg” alt=”yellow banana”/> attributet src beskriver vilken bild som ska finns i bildelementet och attributet alt visar texten som det innehåller ifall bilden kunde inte visas eller för att beskriva bilden för dem med funktionsnedsättning och kan inte se bilden.

2. Skapa en lista med elementen a, img och ol på varsin rad. Under var och en av dem lägger du in en underlista med tre exempel på attribut som finns för dem. Du ska endast ta med de attribut som finns i HTML5. (Tips: titta på referenslistan på W3Schools.)

Svar:

  • <a ><a />, exempel på attribut:
    1. href: anger länkens destination
    2. target: anger var ska länken öppnas i webbläsaren
    3. download: laddar ner filen när användaren trycker på länken
  • <img />, exempel på attribut:
    1. src: anger vilken bild ska visas och det är ett attribut som man måste ha inom img
    2. alt: anger vilken text ska visas ifall det inte fungerar att visa bilden, alternativ
    3. width: anger bildens bredd
    4. height: anger bildens höjd
  • <ol ><ol />, exempel på attribut:
    1. type: anger vilken typ av markör ska visas bredvid listans element
    2. start: anger ordnings nummer for startvärdet. Ex: om vi angav "a" som är "type", för att få listan att börja med b istället för a så måste man ange "2" som "start"
    3. reversed: anger att listan ska vara i omvänd ordning
